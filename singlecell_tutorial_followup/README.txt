Seurat Tutorial with Personal Annotations

This R script follows the official Seurat Guided Clustering Tutorial for the analysis of single-cell RNA sequencing (scRNA-seq) data from PBMC samples. The workflow includes data loading, quality control, normalization, dimensionality reduction, clustering, and visualization.

The code has been adapted with my own comments and annotations as part of my learning process in single-cell transcriptomics and bioinformatics.

Reference:

Hao, Y. et al. Seurat - Guided Clustering Tutorial. Available at: https://satijalab.org/seurat/articles/pbmc3k_tutorial.html
